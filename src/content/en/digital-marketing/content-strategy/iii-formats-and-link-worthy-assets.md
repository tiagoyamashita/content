Formats & link-worthy assets
Some formats **rank**; some **convert**; some **earn links**. Match format to goal — flagship assets power [Link building](../link-building/iii-white-hat-strategies.md).

## 1. Format matrix

| Format | SEO | Conversion | Link earning |
|--------|-----|------------|--------------|
| **Definitive guide** | High | Medium | High |
| **How-to tutorial** | High | Medium | Medium |
| **Comparison / vs** | High (commercial) | High | Medium |
| **Case study** | Medium | High | Medium |
| **Original research / survey** | Medium | Medium | **Very high** |
| **Free tool / calculator** | Medium | High | High |
| **Template / checklist** | Medium | High | High |
| **News / opinion** | Low–medium | Low | Low |

## 2. Link-worthy asset checklist

From [White-hat strategies](../link-building/iii-white-hat-strategies.md) — build before outreach:

| Asset | Why others link |
|-------|-----------------|
| **Data study** | Journalists cite numbers |
| **Interactive tool** | Useful + embeddable |
| **Visual reference** | Infographic with embed code |
| **Canonical glossary** | Defines terms in niche |

```text
Weak:  "We're excited to announce our rebrand"
Strong: "2026 benchmark: median SaaS trial-to-paid rate by segment"
```

## 3. Production tiers (team size)

| Tier | Output | Example |
|------|--------|---------|
| **Solo founder** | 1 pillar + 2 spokes / quarter | Deep guide + tutorials |
| **Small team** | Pillar + monthly spokes + 1 asset / half-year | Guide + annual survey |
| **Content team** | Full cluster + programmatic pages where valid | Hub + tool + comparisons |

Quality and updates beat raw volume.

## 4. E-E-A-T signals (experience, expertise, authority, trust)

Especially for YMYL topics:

| Signal | On-page |
|--------|---------|
| **Byline + bio** | Real author with credentials |
| **Sources cited** | Primary data, official docs |
| **Updated date** | Visible when content refreshed |
| **Contact / about** | Legitimate business presence |

## 5. Repurposing one asset many ways

One research project →

```text
Full report (PDF + landing page)
  →  blog summary
  →  5 chart graphics for social
  →  newsletter edition
  →  pitch to journalists
```

Details: [Distribution & repurposing](v-distribution-and-repurposing.md).

## 6. AI-assisted content (guardrails)

| OK | Risky |
|----|-------|
| Outline, draft, edit with human expert | Publish unreviewed generic AI fluff |
| Update stats and examples | Fabricated citations or quotes |
| Personalize for ICP | Mass duplicate pages for SEO only |

Google rewards **helpful, people-first** content — not production volume alone.
