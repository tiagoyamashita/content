Keywords & search intent
**Keywords** are the phrases people type; **intent** is what they want when they type them. Rankings follow intent match more than keyword count.

## 1. Query types

| Type | User goal | Content shape |
|------|-----------|---------------|
| **Informational** | Learn | How-to, definition, tutorial |
| **Commercial investigation** | Compare before buy | "Best X", "X vs Y", reviews |
| **Transactional** | Buy / sign up | Pricing, product, checkout |
| **Navigational** | Find a brand/site | Homepage, login, docs index |

Classify target queries **before** drafting — a transactional query on a blog post usually underperforms.

## 2. Keyword research (practical)

```text
Seed topic  →  autocomplete, Search Console queries, competitor URLs
  →  cluster by intent  →  assign one primary URL per cluster
```

| Free / low-cost source | What you get |
|------------------------|--------------|
| **Google Search Console** | Queries you already appear for |
| **Autocomplete / PAA** | Related questions |
| **`site:competitor.com topic`** | Pages they invested in |
| **Ahrefs / Semrush** (paid) | Volume, difficulty, gaps |

Early sites: prioritize **low-competition, specific** queries (long-tail) where you can be the best result.

## 3. Keyword mapping (one page, one primary)

| Target keyword cluster | Primary URL | Intent |
|------------------------|-------------|--------|
| "link building guide" | `/link-building-guide/` | Informational |
| "acme pricing" | `/pricing/` | Transactional |
| "acme vs competitor" | `/compare/competitor/` | Commercial |

Avoid two URLs fighting for the same primary query — **keyword cannibalization**.

## 4. Cannibalization: detect and fix

| Symptom | Likely cause |
|---------|--------------|
| Two blog posts swap rankings | Same intent, overlapping content |
| Wrong page ranks | Stronger URL is generic; specific page is weak |

**Fixes:** merge pages (301 weaker → stronger), differentiate intent clearly, or consolidate with a canonical if duplicates are technical.

## 5. Search intent analysis (manual)

For a target query, open top 5–10 results:

| Observe | Decide |
|---------|--------|
| Format (guide vs list vs tool vs video) | Match or beat format |
| Depth (word count, examples, freshness) | Minimum bar to compete |
| SERP features (snippet, PAA, video) | Structure for snippet opportunity |

If all results are listicles, a pure opinion essay may not rank without a different angle.

## 6. Secondary keywords and synonyms

Use **natural variations** in H2s and body — not repetition for its own sake.

```text
Primary: "on-page SEO"
Related: title tags, meta descriptions, internal linking, crawlability
```

Semantic coverage beats exact-match density.

## 7. Local and branded queries

| Query type | On-page note |
|------------|--------------|
| **Branded** | Homepage + dedicated pages rank; protect brand in titles |
| **Local** ("plumber austin") | Location pages, Google Business Profile, NAP consistency |
