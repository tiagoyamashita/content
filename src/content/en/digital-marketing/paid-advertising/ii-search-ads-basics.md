Search ads basics
**Search ads** (Google Ads, Microsoft Ads) show when users query keywords you bid on. High intent — but competition drives **CPC** up fast without tight structure.

## 1. Account structure

```text
Campaign (budget, geo, network)
  └── Ad group (theme of keywords)
        └── Ads + keywords + landing page
```

| Level | Decision |
|-------|----------|
| **Campaign** | Search vs Performance Max, daily budget, location |
| **Ad group** | Tight keyword theme — "link building tools" not everything SEO |
| **Ad** | Headlines, descriptions, extensions |
| **Landing page** | One primary URL per ad group intent |

## 2. Keyword match types

| Type | Syntax | Behavior |
|------|--------|----------|
| **Broad** | keyword | Widest — Google interprets related queries |
| **Phrase** | "keyword" | Query must include phrase meaning |
| **Exact** | [keyword] | Close variants only |

Start ** tighter** (phrase/exact) on limited budget; expand after search term review.

## 3. Search terms report (weekly)

Review **actual queries** that triggered ads:

| Query type | Action |
|------------|--------|
| Irrelevant | Add as **negative keyword** |
| High-converting | Add as exact keyword; maybe dedicated ad |
| Informational only | Negative or separate low-bid campaign |

Negatives prevent budget bleed.

## 4. Quality Score (Google)

Approximate inputs:

| Factor | Improve by |
|--------|------------|
| **Expected CTR** | Relevant ad copy |
| **Ad relevance** | Keywords match ad text |
| **Landing page experience** | Fast, mobile, intent match |

Higher Quality Score → lower CPC for same position.

## 5. Ad copy basics

| Element | Tip |
|---------|-----|
| **Headlines (3+)** | Include keyword theme + benefit |
| **Descriptions** | Proof, CTA, objection |
| **Extensions** | Sitelinks, callouts, structured snippets |

Mirror language from [Positioning](../foundations/iii-audience-and-positioning.md).

## 6. Landing page alignment

| Ad promise | Landing page |
|------------|--------------|
| "Free trial — no card" | Above-fold signup, same wording |
| "Compare plans" | Pricing table, not homepage |
| "Download guide" | Form + guide — not blog index |

Mismatch kills **conversion rate** and Quality Score.

## 7. Bidding starter

| Goal | Bid strategy (starter) |
|------|------------------------|
| Learn | Manual CPC or Max clicks with low cap |
| Leads | Max conversions (with conversion tracking) |
| Revenue | Target ROAS (e-commerce, enough volume) |

Details: [Budget, bidding & ROAS](iv-budget-bidding-and-roas.md).
