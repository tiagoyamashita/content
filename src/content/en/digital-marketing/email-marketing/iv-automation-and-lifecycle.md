Automation & lifecycle
**Automation** sends the right message when a user hits a trigger — welcome, - onboarding, nurture, win-back — without manual sends each time.

## 1. Core lifecycle flows

```text
Subscribe  →  welcome series
Sign up    →  onboarding tips
Trial day 7 →  upgrade nudge
Inactive   →  re-engagement
Customer   →  expansion / referral ask
```

Map flows to [funnel stages](../foundations/ii-channels-and-funnel.md).

## 2. Welcome series (3–5 emails)

| Email | Timing | Content |
|-------|--------|---------|
| 1 | Immediate | Deliver magnet; set expectations |
| 2 | Day 2 | Best beginner resource |
| 3 | Day 5 | Social proof / story |
| 4 | Day 7 | Soft CTA to product or deep guide |

Stop series if they already converted.

## 3. Triggers and rules

| Trigger | Example automation |
|---------|-------------------|
| **Tag added** | "Interested in SEO" → SEO tip track |
| **Event** | Downloaded whitepaper → follow-up case study |
| **Date** | Trial ending in 3 days |
| **Inactivity** | No open 90 days → win-back |

Define **exit conditions** — purchased, unsubscribed, replied.

## 4. Transactional vs marketing automation

| | Transactional | Marketing automation |
|---|---------------|---------------------|
| **Opt-in** | Implied by account | Explicit marketing consent |
| **Content** | Receipt, security alert | Promos, newsletter |
| **Unsubscribe** | N/A (account emails) | Required link |

Don't hide marketing in password-reset emails.

## 5. E-commerce extras

| Flow | Purpose |
|------|---------|
| **Cart abandon** | Reminder + optional incentive |
| **Post-purchase** | Thank you, review request, cross-sell |
| **Browse abandon** | Viewed product, no cart — use carefully |

Test timing — first abandon email at 1–4 hours often works.

## 6. Build order (startup)

```text
1. Welcome series (new subscribers)
2. One onboarding path (new signups)
3. Re-engagement (inactive 90+ days)
4. Everything else
```

Ship flow 1 before optimizing subject lines on broadcast newsletters.
