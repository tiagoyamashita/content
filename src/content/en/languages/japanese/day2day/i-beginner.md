Verbs & office replies — connect your words
You already know **nouns** from [Frequent IT kanji](../beginner-alphabet/iv-frequent-it-kanji.md) and [Common words](../vocabulary/i-common-words.md). This page shows how to attach **simple verbs** in **〜します** form — the polite style used in tickets, email, Slack, and docs at work.

Previous: [Hiragana](../beginner-alphabet/iii-hiragana.md) · [Overview](../beginner-alphabet/i-overview.md)

Each table includes **katakana** beside Japanese so you can read phrases aloud even if hiragana is still new.

## The main pattern: noun + します

Many IT terms are **nouns**. Turn them into verbs by adding **します** (polite “do”).

| Pattern | Katakana | Meaning |
|---------|----------|---------|
| 設定 + します | セッテイシマス | configure / set up |
| 確認 + します | カクニンシマス | confirm |
| 更新 + します | コウシンシマス | update |
| 削除 + します | サクジョシマス | delete |
| 保存 + します | ホゾンシマス | save |
| 実行 + します | ジッコウシマス | execute / run |
| 登録 + します | トウロクシマス | register |
| 接続 + します | セツゾクシマス | connect |
| 変更 + します | ヘンコウシマス | change |
| 追加 + します | ツイカシマス | add |
| 作成 + します | サクセイシマス | create |
| 実装 + します | ジッソウシマス | implement |
| 検証 + します | ケンショウシマス | verify / validate |
| 承認 + します | ショウニンシマス | approve |
| レビュー + します | レビューシマス | review |
| デプロイ + します | デプロイシマス | deploy |

**Tip:** Know the noun from the common-words list → add **します** for the verb you will read at work.

## Core verbs in 〜ます form

Pair these with words you already know. All examples use polite **〜ます**.

| Verb | Katakana | English | Example with your vocab |
|------|----------|---------|---------------------------|
| します | シマス | do | 設定します — セッテイシマス (configure) |
| なります | ナリマス | become | 完了になります — カンリョウニナリマス (becomes complete) |
| できます | デキマス | can do | 接続できます — セツゾクデキマス (can connect) |
| 使います | ツカイマス | use | ライブラリを使います — ライブラリオツカイマス (use a library) |
| 見ます | ミマス | see / view | 画面を見ます — ガメンヲミマス (view the screen) |
| 読みます | ヨミマス | read | ログを読みます — ログヲヨミマス (read the log) |
| 書きます | カキマス | write | コードを書きます — コードヲカキマス (write code) |
| 入れます | イレマス | put in | 値を入れます — アタイヲイレマス (enter a value) |
| 出します | ダシマス | put out / output | 結果を出します — ケッカヲダシマス (output results) |
| 送ります | オクリマス | send | リクエストを送ります — リクエストオオクリマス (send a request) |
| 受け取ります | ウケトリマス | receive | レスポンスを受け取ります — レスポンスウケトリマス (receive a response) |
| 開きます | ヒラキマス | open | ファイルを開きます — ファイルオヒラキマス (open a file) |
| 閉じます | トジマス | close | 接続を閉じます — セツゾクオトジマス (close the connection) |
| 始めます | ハジメマス | start | 処理を始めます — ショリオハジメマス (start processing) |
| 終わります | オワリマス | end | バッチが終わります — バッチガオワリマス (the batch ends) |
| 止めます | トメマス | stop | サーバーを止めます — サーバーオトメマス (stop the server) |
| 待ちます | マチマス | wait | 応答を待ちます — オウトウオマチマス (wait for a response) |
| 含みます | フクミマス | include | エラーを含みます — エラーオフクミマス (includes an error) |
| 必要です | ヒツヨウデス | need (noun/adj.) | 設定が必要です — セッテイガヒツヨウデス (settings are required) |

## Variations on 〜します

Same verb root — different endings you will see every day.

| Form | Katakana | When it appears |
|------|----------|-----------------|
| 〜します | 〜シマス | present / future — 設定します — セッテイシマス (I/we will configure) |
| 〜しました | 〜シマシタ | past — 更新しました — コウシンシマシタ (updated) |
| 〜してください | 〜シテクダサイ | please do — 確認してください — カクニンシテクダサイ (please confirm) |
| 〜できません | 〜デキマセン | cannot — 接続できません — セツゾクデキマセン (cannot connect) |
| 〜されています | 〜サレテイマス | is being / has been — 処理されています — ショリサレテイマス (is being processed) |
| 〜となります | 〜トナリマス | formal “will be” — 完了となります — カンリョウトナリマス (will be completed) |
| 〜について | 〜ニツイテ | about / regarding — 障害について — ショウガイニツイテ (about the incident) |

## Particles: the small glue words

You do not need all grammar — just these four for IT phrases.

| Particle | Katakana | Role | Example |
|----------|----------|------|---------|
| を | ヲ | object marker | 設定を保存します — セッテイヲホゾンシマス |
| が | ガ | subject marker | エラーが発生しました — エラーガハッセイシマシタ |
| に | ニ | to / in / at | 本番にデプロイします — ホンバンニデプロイシマス |
| の | ノ | possession / link | サーバーの設定 — サーバーノセッテイ |

## Office replies — positive (yes / OK)

Short replies you will hear and use in meetings, Slack, and email. Most use **〜ます** or **〜しました**.

| Japanese | Katakana | English | When to use |
|----------|----------|---------|-------------|
| はい、承知しました | ハイ、ショウチシマシタ | Yes, understood | Default “got it” after a request |
| 了解です | リョウカイデス | Got it / roger | Slack, quick chat — slightly casual but common |
| 了解しました | リョウカイシマシタ | Understood | Same as above, a bit more formal |
| かしこまりました | カシコマリマシタ | Certainly / understood | Formal; clients, vendors, senior staff |
| 問題ありません | モンダイアリマセン | No problem | You agree; the plan is fine |
| 大丈夫です | ダイジョウブデス | It's OK / fine | Reassuring; no issue found |
| 可能です | カノウデス | It's possible / yes, we can | Answering “can we do X?” |
| 対応します | タイオウシマス | I'll handle it / we will address it | You take ownership of the task |
| 進めます | ススメマス | I'll proceed | You will start or continue work |
| 確認しました | カクニンシマシタ | I checked / confirmed | After looking at logs, code, or a doc |
| 完了しました | カンリョウシマシタ | Completed | Task or fix is done |
| 修正しました | シュウセイシマシタ | Fixed / corrected | Bug or doc updated |
| ありがとうございます | アリガトウゴザイマス | Thank you | After help or a review |
| よろしくお願いします | ヨロシクオネガイシマス | Please take care of it / thanks in advance | Start of thread or when asking a favor |
| お疲れ様です | オツカレサマデス | Good work / hello-goodbye | Daily greeting in the office |
| 賛成です | サンセイデス | I agree | Reviews, design meetings |
| 問題ないと思います | モンダイナイトオモイマス | I don't think there's an issue | Soft yes in a review |

## Office replies — negative (no / not yet / blocked)

Japanese offices often **soften “no”**. Listen for these instead of a blunt いいえ.

| Japanese | Katakana | English | When to use |
|----------|----------|---------|-------------|
| すみません、難しいです | スミマセン、ムズカシイデス | Sorry, that's difficult | Soft no — deadline, scope, or tech blocker |
| ちょっと難しいです | チョットムズカシイデス | That's a bit difficult | Softer still; room to negotiate |
| 確認します | カクニンシマス | I'll check | Buys time; not yes yet |
| 確認が必要です | カクニンガヒツヨウデス | Confirmation is needed | Missing info; can't decide now |
| 検討します | ケントウシマス | I'll consider it | Polite deferral — not a yes |
| できません | デキマセン | Cannot (do it) | Clear inability — tech or permission |
| 不可能です | フカノウデス | Impossible / not feasible | Stronger than できません |
| 問題があります | モンダイガアリマス | There is a problem | Something is wrong |
| エラーが出ています | エラーガデテイマス | An error is occurring | Logs, screen, or test failure |
| まだ完了していません | マダカンリョウシテイマセン | Not completed yet | Status update — work in progress |
| 時間がかかります | ジカンガカカリマス | It will take time | Push back on urgent requests |
| 少々お待ちください | ショウショウオマチクダサイ | Please wait a moment | Checking now; formal |
| 別の方法を検討します | ベツノホウホウヲケントウシマス | We'll consider another approach | Alternative proposal after a no |
| 申し訳ありません | モウシワケアリマセン | I'm sorry | Apology before bad news |
| 恐れ入りますが、〜 | オソレイリマスガ、〜 | I'm afraid… (before bad news) | Formal email opener for a negative reply |
| 承知しかねます | ショウチシカネマス | Difficult to accept | Formal, firm no — rare but important |

**Pattern:** すみません — スミマセン + reason + 確認します — カクニンシマス / 難しいです — ムズカシイデス is more natural at work than a plain いいえ — イイエ.

## Practice drill

Pick any noun from [Common words](../vocabulary/i-common-words.md), then:

1. Add **します** — シマス → verb (`承認` → 承認します — ショウニンシマス).
2. Wrap in **してください** — シテクダサイ → polite request (`確認してください` — カクニンシテクダサイ).
3. Swap in **できません** — デキマセン → failure message (`保存できません` — ホゾンデキマセン).
4. Reply like at the office — **承知しました** — ショウチシマシタ (yes) or **確認します** — カクニンシマス (not yet / soft no).

If the sentence still makes sense in English, you are reading real Japanese IT text — not memorizing isolated words.

## How Japanese usually answer at work

A quick guide to **what to expect** and **how to sound natural** — not a full culture lesson, just patterns you will see every day in engineering teams.

### 1. Acknowledge first, act second

When someone asks for something, the reply often **starts with acceptance of the message**, not the answer itself.

| Flow | Japanese | Katakana |
|------|----------|----------|
| Request → ack | 承知しました。確認します。 | ショウチシマシタ。カクニンシマス。 |
| Request → ack + timeline | 承知しました。午後対応します。 | ショウチシマシタ。ゴゴタイオウシマス。 |
| Bad news → apology first | 申し訳ありません。まだ完了していません。 | モウシワケアリマセン。マダカンリョウシテイマセン。 |

Western replies often jump straight to the answer. In Japan, **承知しました** / **了解です** first signals “I heard you” — even when the real answer is “not yet.”

### 2. “Yes” is rarely a bare はい

| Situation | Typical reply | Katakana |
|-----------|---------------|----------|
| Task assigned | 承知しました。対応します。 | ショウチシマシタ。タイオウシマス。 |
| Can you do it? | 可能です。 / 問題ありません。 | カノウデス。 / モンダイアリマセン。 |
| Agree in meeting | 賛成です。 / 問題ないと思います。 | サンセイデス。 / モンダイナイトオモイマス。 |
| Done | 完了しました。 / 修正しました。 | カンリョウシマシタ。 / シュウセイシマシタ。 |

**はい** alone is polite but **incomplete** at work. Pair it with **what you will do** or **what happened**.

### 3. “No” is usually indirect

Direct **いいえ** / **できません** (alone) can sound harsh. Common substitutes:

| Meaning | Japanese | Katakana |
|---------|----------|----------|
| Not yet | 確認します。 / 少々お待ちください。 | カクニンシマス。 / ショウショウオマチクダサイ。 |
| Probably no | ちょっと難しいです。 / 時間がかかります。 | チョットムズカシイデス。 / ジカンガカカリマス。 |
| Need approval | 確認が必要です。 / 上長に確認します。 | カクニンガヒツヨウデス。 / ジョウチョウニカクニンシマス。 |
| Firm no (formal) | 恐れ入りますが、対応できかねます。 | オソレイリマスガ、タイオウデキカネマス。 |

Listen for **難しい** and **確認** — they often mean “no for now” or “I need time,” not “I'm confused.”

### 4. Status updates are proactive

Silence until done is less common than in some cultures. Expect (and send) short updates:

- **対応中です。** — タイオウチュウデス — working on it
- **確認中です。** — カクニンチュウデス — investigating
- **完了しました。** — カンリョウシマシタ — done
- **問題ありませんでした。** — モンダイアリマセンデシタ — checked; all clear

In Slack or tickets, a one-line **確認しました** or **対応します** closes the loop fast.

### 5. Email and chat shape

A typical written reply follows this skeleton:

1. **Greeting** — お疲れ様です。 — オツカレサマデス / お世話になっております。 — オセワニナッテオリマス
2. **Thanks or apology** — ご連絡ありがとうございます。 — ゴレンラクアリガトウゴザイマス / 申し訳ありません。 — モウシワケアリマセン
3. **Body** — facts, します / しました verbs, linked to your vocab
4. **Close** — よろしくお願いします。 — ヨロシクオネガイシマス / 以上、よろしくお願いいたします。 — イジョウ、ヨロシクオネガイイタシマス

You do not need perfect keigo on day one — **consistent polite ます form** plus **承知しました** / **申し訳ありません** goes a long way. For client email, phone, and 尊敬語・謙譲語 tables, see [Keigo](../vocabulary/iii-keigo.md). For full 件名 and 本文 templates, see [Email templates](../vocabulary/iv-email-templates.md).

### 6. One-line cheat sheet

| They ask… | Japanese | Katakana |
|-----------|----------|----------|
| Please do X | 承知しました。対応します。 | ショウチシマシタ。タイオウシマス。 |
| Is it done? | 完了しました。 / まだ完了していません。 | カンリョウシマシタ。 / マダカンリョウシテイマセン。 |
| Can we do X? | 可能です。 / ちょっと難しいです。 | カノウデス。 / チョットムズカシイデス。 |
| What happened? | 確認しました。〜が原因です。 | カクニンシマシタ。〜ガゲンインデス。 |
| Sorry to bother | 問題ありません。 / こちらこそありがとうございます。 | モンダイアリマセン。 / コチラコソアリガトウゴザイマス。 |

When in doubt: **acknowledge → verb in ます form → brief reason if needed.** That matches how most Japanese engineers answer in the office.
