## Chapter 3 — First common kanji

**Goal:** Recognize the kanji you see most often before diving into full IT vocabulary.

One table per group. Each row: **Kanji · English meaning · Example compound (reading — gloss)**.

Study **in this order**:

### 3.1 Calendar

Words for calendars, scheduling, and time blocks.

| Kanji | English |
|-------|---------|
| 年 | year |
| 月 | month |
| 日 | day |
| 週 | week |
| 曜 | weekday (in 曜日) |
| 暦 | calendar |
| 季 | season / quarter |
| 予 | plan / schedule |
| 祝 | holiday (celebrate) |
| 休 | rest / day off |
| 締 | cutoff / closing date |

### 3.2 Dates

Relative dates, clock time, and deadlines.

| Kanji | English |
|-------|---------|
| 今 | now / current |
| 昨 | previous (yesterday, last year) |
| 明 | next (tomorrow) |
| 来 | coming / next |
| 毎 | every |
| 午 | noon (AM/PM) |
| 朝 | morning |
| 昼 | midday |
| 夕 | evening |
| 夜 | night |
| 時 | time / hour |
| 間 | interval / span |
| 秒 | second |
| 半 | half |
| 番 | number / turn |

### 3.3 Weekday names

The seven-day row uses five elements + 日 + 月:

| Kanji | English | Day |
|-------|---------|-----|
| 日 | sun | Sunday |
| 月 | moon | Monday |
| 火 | fire | Tuesday |
| 水 | water | Wednesday |
| 木 | tree | Thursday |
| 金 | gold | Friday |
| 土 | earth | Saturday |

Example compounds: 日曜日 (にちようび — Sunday), 金曜日 (きんようび — Friday).

### 3.4 Documents related

Kanji from 基本設計書 and 詳細設計書 — titles, sections, and review flow.

| Kanji | English |
|-------|---------|
| 本 | main / production / book |
| 基 | foundation |
| 詳 | detailed |
| 細 | detail |
| 設 | design |
| 計 | plan |
| 書 | document |
| 概 | outline |
| 背 | background |
| 景 | context / scene |
| 的 | purpose / target |
| 業 | business |
| 務 | task / duty |
| 対 | target |
| 範 | scope |
| 方 | direction / method |
| 針 | policy |
| 用 | use |
| 語 | terminology |
| 仕 | specification |
| 様 | format / form |
| 説 | explanation |
| 図 | diagram |
| 遷 | transition |
| 状 | state |
| 態 | condition |
| 流 | flow |
| 課 | issue |
| 題 | topic / problem |

Key document names to recognize:

- 基本設計書 (きほんせっけいしょ — basic design document)
- 詳細設計書 (しょうさいせっけいしょ — detailed design document)
- 要件定義書 · 仕様書 · テスト仕様書 (see [Common words](../ii-common-words.md))
