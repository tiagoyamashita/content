Office & daily — conversations
Morning **挨拶**, hallway talk, and **desk-side** requests. Mostly **丁寧語**; **お疲れ様** is standard — never **ご苦労様** upward. Keigo reference: [Keigo](../iii-keigo.md).

---

## 1. Arriving at the office

**Context:** You arrive; colleague already at desk. **Internal.**

| | |
|---|---|
| **同僚（田中）** | おはようございます。 |
| | (ohayō gozaimasu) — Good morning. |
| | [丁寧: standard morning greeting] |
| **あなた** | おはようございます。昨日のデプロイ、大丈夫でしたか。 |
| | — Good morning. Was yesterday's deploy OK? |
| **田中** | ええ、問題なかったです。お疲れ様でした。 |
| | — Yes, no issues. Thanks for the work (yesterday). |
| **あなた** | よかったです。本日もよろしくお願いします。 |
| | — Glad to hear it. Let's have a good day today. |

**Note:** **お疲れ様でした** for work already done (yesterday's release). **お疲れ様です** for ongoing / today.

---

## 2. Passing 部長 in the hallway

**Context:** Brief encounter. **Internal — senior.**

| | |
|---|---|
| **あなた** | 部長、お疲れ様です。 |
| | — Good afternoon / hello, Director. |
| **田村部長** | お疲れ様。〇〇の件、進んでる？ |
| | — Hey. How's the XX matter going? (casual question from senior) |
| **あなた** | はい、本日中にテストを完了する予定です。終わり次第、ご報告いたします。 |
| | — Yes, I plan to finish testing today. I'll report when done. |
| | [謙譲: **ご報告いたします** — humble "I will report"] |
| **田村部長** | わかった。よろしく。 |
| | — Got it. Thanks. |

## 3. Leaving before others (お先に)

**Context:** You leave at scheduled time; team still working. **Internal.**

| | |
|---|---|
| **あなた** | お先に失礼します。本日の対応はチケット【1234】に記載しました。 |
| | — Excuse me for leaving first. Today's work is on ticket 1234. |
| | [丁寧: **お先に失礼します** — fixed phrase] |
| **鈴木課長** | お疲れ様。明日の朝会、9時からね。 |
| | — Good work. Standup is 9 tomorrow. |
| **あなた** | 承知しました。失礼します。 |
| | — Understood. Goodbye. |

## 4. Asking a colleague for a quick favor

**Context:** Same team; Slack is also fine — this is spoken at desk. **Internal.**

| | |
|---|---|
| **あなた** | すみません、ちょっとよろしいですか。ステージングのログ、見ていただけますか。 |
| | — Sorry, do you have a moment? Could you look at the staging logs? |
| | [丁寧: **見ていただけますか** — polite request] |
| **田中** | いいですよ。どの時間帯？ |
| | — Sure. Which time range? |
| **あなた** | 10時から11時です。お忙しいところ恐縮ですが、よろしくお願いします。 |
| | — 10 to 11. Sorry to bother you — thanks in advance. |
| **田中** | 了解。あとで連絡するね。 |
| | — OK. I'll get back to you. |

## 5. Visitor at reception (you fetch a guest)

**Context:** Client arrived; you escort from lobby. **External — client on site.**

| | |
|---|---|
| **あなた** | 佐藤様、お待たせいたしました。◇◇の□□でございます。 |
| | — Mr. Sato, sorry to keep you waiting. I'm XX from ◇◇. |
| | [謙譲: **お待たせいたしました** — sorry for the wait] |
| **佐藤様** | お世話になっております。よろしくお願いします。 |
| | — Thank you for your support. Pleased to meet you. |
| **あなた** | こちらこそお世話になっております。会議室へご案内いたします。 |
| | — Likewise. I'll show you to the meeting room. |
| | [謙譲: **ご案内いたします** — humble "I will guide"] |
| **佐藤様** | よろしくお願いします。 |
| **あなた** | こちらへどうぞ。お荷物、お預かりしましょうか。 |
| | — This way, please. Shall I take your bag? |

## 6. Tea / break room small talk

**Context:** Low stakes; still avoid blunt plain form. **Internal.**

| | |
|---|---|
| **同僚** | お疲れ様。ちょっと休憩しない？ |
| | — Good work. Want a short break? |
| **あなた** | お疲れ様です。はい、5分だけなら大丈夫です。 |
| | — Thanks. Yes, five minutes is fine. |
| **同僚** | さっきの会議、長かったね。 |
| | — That meeting was long, huh. |
| **あなた** | そうですね。資料は後で共有されますね。 |
| | — Yeah. They'll share the deck later, right? |

## Keigo checklist (this file)

| Phrase | Type | When |
|--------|------|------|
| お疲れ様です | 丁寧 (+ fixed) | Default greeting |
| ご苦労様です | ❌ upward | To subordinates only — not 部長 |
| お先に失礼します | 丁寧 | Leaving office |
| ご報告いたします | 謙譲 | Reporting to senior |
| ご案内いたします | 謙譲 | Escorting client |
| お待たせいたしました | 謙譲 | Client waited |
