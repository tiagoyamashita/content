Phone & client calls — conversations
**電話** scripts for answering, transferring, and calling **顧客** or **ベンダー**. Phone keigo is often more formal than Slack. Phrase list: [Keigo](../iii-keigo.md) §5f.

---

## 1. Answering an internal call

**Context:** Colleague calls your desk. **Internal.**

| | |
|---|---|
| **あなた** | はい、開発部の□□です。 |
| | — Yes, this is XX from Development. |
| **田中** | お疲れ様。ログ、もう見れた？ |
| **あなた** | お疲れ様です。はい、確認しました。原因はキャッシュです。 |
| | — Yes, I checked. Cause is the cache. |
| **田中** | 了解。ありがとう。 |
| **あなた** | どういたしまして。失礼します。 |
| | — You're welcome. Bye. |

## 2. Answering external call (company line)

**Context:** Client calls main number; you pick up. **External.**

| | |
|---|---|
| **あなた** | お電話ありがとうございます。◇◇株式会社でございます。 |
| | — Thank you for calling. This is ◇◇ Corporation. |
| | [丁寧: **でございます** — formal copula] |
| **佐藤様** | 佐藤と申します。昨日のリリースについて確認したいのですが。 |
| | — This is Sato. I'd like to confirm yesterday's release. |
| | [謙譲: **と申します** — humble self-intro] |
| **あなた** | 佐藤様、お世話になっております。担当の□□が承ります。 |
| | — Mr. Sato, thank you for your business. I'll assist you. |
| | [謙譲: **承ります** — humble "I receive your call/request"] |
| **あなた** | 恐れ入りますが、内容を確認させていただいてもよろしいでしょうか。 |
| | — May I confirm the details? |
| **佐藤様** | ええ、ログインできないユーザーがいるんです。 |
| **あなた** | かしこまりました。ただいま担当者に確認いたしますので、少々お待ちください。 |
| | — Certainly. I'll check with the person in charge — please hold. |
| | [丁寧+: **かしこまりました**; 謙譲: **確認いたします**] |

## 3. Transferring to senior (escalation on phone)

**Context:** Client needs 部長. **External.**

| | |
|---|---|
| **あなた** | 少々お待ちください。担当の部長に確認いたします。 |
| | — Please hold. I'll check with the director in charge. |
| **あなた** | （部長へ）田村部長、お疲れ様です。○○様の佐藤様からお電話です。ログイン障害のご連絡です。 |
| | — Director Tamura, call from Mr. Sato at XX re login outage. |
| **田村部長** | わかった。つないで。 |
| **あなた** | （佐藤様へ）お待たせいたしました。部長の田村が承ります。お電話変わりました。 |
| | — Sorry for the wait. Director Tamura will take the call. You're now connected. |
| | [謙譲: **お待たせいたしました**; fixed: **お電話変わりました**] |
| **田村部長** | 佐藤様、お世話になっております。田村でございます。 |
| **佐藤様** | お世話になっております。状況をお聞かせいただけますか。 |
| | — Please tell me the situation. |
| | [尊敬: **お聞かせいただけますか** — humble request form from client] |

## 4. Outbound call to client (you call them)

**Context:** Follow up after email. **External — you initiate.**

| | |
|---|---|
| **あなた** | お世話になっております。◇◇の□□と申します。 |
| | — Thank you for your business. This is XX from ◇◇. |
| **あなた** | 先ほどメールいただきました件で、ご連絡させていただきました。 |
| | — I'm calling about the email you sent earlier. |
| | [謙譲: **ご連絡させていただきました**] |
| **佐藤様** | ああ、□□さん。お忙しいところありがとうございます。 |
| **あなた** | とんでもございません。ご確認いただきたい点が一点ございます。 |
| | — Not at all. There's one point we'd like you to confirm. |
| | [尊敬: **ご確認いただきたい** — their confirmation elevated] |
| **佐藤様** | はい、どうぞ。 |
| **あなた** | リリース日は3月15日のままでよろしいでしょうか。 |
| **佐藤様** | はい、問題ありません。 |
| **あなた** | ありがとうございます。では、その日程で進めさせていただきます。 |
| | — Thank you. We'll proceed on that schedule. |
| **あなた** | 失礼いたします。 |
| | — Goodbye (formal). |

## 5. Vendor support hotline

**Context:** You call cloud vendor. **External — they use humble toward you.**

| | |
|---|---|
| **オペレーター** | お電話ありがとうございます。○○サポートでございます。 |
| **あなた** | お世話になっております。◇◇の□□です。APIのエラーについて問い合わせたいです。 |
| **オペレーター** | かしこまりました。インシデントIDはお分かりでしょうか。 |
| | — Certainly. Do you know the incident ID? |
| **あなた** | はい、INC-98765です。 |
| **オペレーター** | 確認いたします。少々お待ちください。 |
| | — I'll check. Please hold. [vendor humble] |
| **オペレーター** | お待たせいたしました。現在、調査中でございます。 |
| **あなた** | 承知しました。結果が分かり次第、ご連絡いただけますか。 |
| | — Understood. Could you contact us when you know? |
| | [尊敬: **ご連絡いただけますか** — their action elevated] |

| | |
|---|---|
| **あなた** | 申し訳ございません。お声が遠いようです。もう一度お願いいたします。 |
| | — Sorry. Your voice sounds far. Could you repeat? |
| **佐藤様** | （ repeats ） |
| **あなた** | ありがとうございます。承知いたしました。 |
| | — Thank you. Understood. |
| | [謙譲: **承知いたしました** — formal ack to client] |
