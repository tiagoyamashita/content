Keigo — honorific Japanese for work
**敬語 (keigo)** is honorific language used with **clients**, **senior staff**, **vendors**, and in **formal email**. At Japanese IT companies you will hear it in meetings, phone calls, and customer-facing tickets. This page is a **reference list with examples** — pair with [Verbs & office replies](../day2day/i-beginner.md) for daily ます-form patterns, [Daily conversations](daily-conversations/i-overview.md) for full situational scripts, and [Corporate structure](ii-corporate-structure.md) for titles (部長, 課長).

## The three types

| Type | Japanese | Who is elevated | Typical use |
|------|----------|-----------------|-------------|
| **Polite** | 丁寧語 (teineigo) | Neutral respect | Default at work: 〜ます / 〜です |
| **Respectful** | 尊敬語 (sonkeigo) | The **other person** (or their actions) | Client did X; boss will attend |
| **Humble** | 謙譲語 (kenjougo) | You **lower yourself** | I will visit; I will report |

```text
Client / boss action     → 尊敬語  (elevate them)
Your action toward them  → 謙譲語  (humble yourself)
Either party, neutral    → 丁寧語  (ます・です)
```

**Rule of thumb for engineers:** internal Slack can stay **丁寧語**; email to **取引先** and calls with **顧客** need **尊敬語 + 謙譲語** on key verbs.

## 1. 丁寧語 (teineigo)

Plain dictionary form → **ます / です**. This is the floor for professional Japanese.

| Plain | Polite | Example |
|-------|--------|---------|
| する | します | 対応**します**。 |
| できる | できます | 本日中に修正**できます**。 |
| ある | あります | 問題が**あります**。 |
| ない | ありません | 障害は**ありません**。 |
| いい | いいです / よろしいです | その方針で**よろしいです**。 |
| だ | です | 担当**です**。 |
| 行く | 行きます | 15時に会議に**行きます**。 |
| 来る | 来ます | 明日**来ます**。 |
| 見る | 見ます | ログを**見ます**。 |
| 言う | 言います | 後ほど**言います**。 |

**Past / negative (patterns you will write daily):**

| Form | Example |
|------|---------|
| 〜ました | 確認**しました**。 |
| 〜ません | まだ完了し**ません**。 |
| 〜ませんでした | 昨日は対応でき**ませんでした**。 |
| 〜でしょうか | ご確認いただけ**でしょうか**。 (soft question) |

### 2b. お / ご ＋ ます-stem ＋ になる

For verbs without a special form:

| Plain | Respectful | Example |
|-------|------------|---------|
| 確認する | ご確認になる | 仕様を**ご確認に**なります。 |
| 利用する | ご利用になる | 本番環境を**ご利用に**なります。 |
| 登録する | ご登録になる | ユーザーが**ご登録に**なりました。 |
| 変更する | ご変更になる | 設定を**ご変更に**なりますか。 |
| 承認する | ご承認になる | 部長が**ご承認に**なりました。 |

**お vs ご:** **ご** before on-yomi (漢語); **お** before kun-yomi and many native words. When unsure at work, **ご** is safer for business compounds (ご確認, ご利用, ご検討).

### 2c. お / ご ＋ ます-stem ＋ なさる

Slightly less common than になる; still appears in formal speech:

| Example | Meaning |
|---------|---------|
| いつ**ご都合なさいます**か。 | When is convenient for you? |
| **ご検討なさい**ますようお願いいたします。 | Please consider (very formal). |

## 3. 謙譲語 (kenjougo) — humble yourself

### 3a. Special humble verbs

| Plain | Humble | Reading | Example |
|-------|--------|---------|---------|
| する | いたす | いたす | 対応**いたします**。 |

| 飲む | いただく | いただく | コーヒーを**いただき**ます。 |

| 聞く | 伺う / 承る | うかがう / うけたまわる | 要件を**伺い**ます。 |
| 来る | 伺う / 参る | うかがう / まいる | 明日**伺います**（お伺いします）。 |
| 行く | 参る / 伺う | まいる / うかがう | 貴社に**参ります**。 |
| 知る | 存じる / 存じ上げる | ぞんじる | **存じ上げません**（I do not know — humble）。 |
| 会う | お目にかかる | おめにかかる | 担当者に**お目にかかり**ました。 |
| 読む | 拝読する | はいどくする | 仕様書を**拝読いたしました**。 |
| 書く | お書きする | おかきする | こちらに**お書きいたします**。 |

| 渡す | お渡しする | おわたしする | 修正版を**お渡しします**。 |
| 送る | お送りする | おおくりする | データを**お送りいたします**。 |

| 使う | お使いする | おつかいする | ツールを**お使いください**（to client: お使いください = respectful imperative）。 |
| 借りる | 拝借する | はいしゃくする | 環境を**拝借いたします**。 |
| 貸す | お貸しする | おかしする | 検証用IDを**お貸しいたします**。 |
| 教える | お教えする | おおしえする | 手順を**お教えいたします**。 |
| 思う | 存じる | ぞんじる | **存じます**（I think — humble）。 |
| あげる | 差し上げる | さしあげる | 成果物を**差し上げます**。 |
| 訪問する | 伺う | うかがう | 本日**伺い**ます。 |
| 連絡する | ご連絡する | ごれんらくする | 後ほど**ご連絡いたします**。 |
| 案内する | ご案内する | ごあんないする | 会議室へ**ご案内いたします**。 |
| 説明する | ご説明する | ごせつめいする | 仕様を**ご説明いたします**。 |
| 確認する | 確認いたす / ご確認する | — | こちらで**確認いたします**。 |
| 対応する | 対応いたす | — | 至急**対応いたします**。 |
| 調査する | 調査いたす | — | 原因を**調査いたします**。 |
| 修正する | 修正いたす | — | 本日中に**修正いたします**。 |

### 3b. お / ご ＋ ます-stem ＋ する

| Plain | Humble | Example |
|-------|--------|---------|
| 連絡する | ご連絡する | 明日**ご連絡いたします**。 |
| 報告する | ご報告する | 結果を**ご報告いたします**。 |
| 相談する | ご相談する | PMに**ご相談いたします**。 |
| 依頼する | お願いする | ご確認を**お願いいたします**。 |
| 質問する | ご質問する | **ご質問いただき**ありがとうございます。 |

### 3c. 〜させていただく (humble permission)

“I humbly do (with your allowance)” — very common in business email:

| Example | Meaning |
|---------|---------|
| 確認**させていただき**ます。 | I will check (humbly). |
| お借り**させていただき**ます。 | I will borrow / use (your time, env). |
| ご説明**させていただき**ます。 | Allow me to explain. |
| ご連絡**させていただき**ます。 | I will contact you (humbly). |
| お時間をいただき**ありがとうございます**。 | Thank you for your time. |

| ご不明な点はございますか | ごふめいな… | Anything unclear? |
| 議事録を共有いたします | ぎじろくを… | I'll share minutes |

### Email to client (incident)

△△様

○○株式会社の□□でございます。

本日10時頃より、貴社ご利用のサービスで障害が発生しており、

現在、原因を調査しております。

お急ぎのお問い合わせは、本メールへご返信ください。

何卒よろしくお願いいたします。

### Slack internal (丁寧語 is enough)

```text
部長、承知しました。15時までに修正します。

```

### Asking boss for approval (尊敬 + 謙譲)

### Vendor to your team (they use humble toward you)

Listen for: **伺います** (I will come), **提案いたします** (we propose), **〜させていただきます**.

## 8. Common mistakes (avoid)

| Mistake | Why wrong | Fix |
|---------|-----------|-----|
| ご苦労様です → 部長 | ご苦労 is **downward** only | お疲れ様です |
| お疲れ様です → お客様 | Too casual for clients | お世話になっております |
| 二重敬語 (ご確認させていただきます + お) | Redundant honorific stacking | ご確認させていただきます |
| いらっしゃる about yourself | Respectful verb for **you** | 参ります / 伺います |
| 参ります about the client | Humble verb for **them** | いらっしゃいます / お越しになります |
| 知りません (client) | Blunt | 存じ上げません / 確認いたします |
| できる？ (client email) | Too casual | 可能でございます / 対応可能です |
| 了解 (only) to client | Slightly casual | 承知いたしました |
| すみません only (major outage) | Too light | 大変申し訳ございません |
| 〜してもらう (about client doing) | Client is not もらう subject casually | 〜していただく |

## 9. Quick decision flow

```text
Who is the subject of the verb?
  ├── Client / boss / senior → 尊敬語 (or 丁寧語 minimum)
  ├── You / your team → 謙譲語 (external) or 丁寧語 (internal)
  └── Written to 取引先? → opener お世話になっております + humble close
```

| Channel | Minimum level |
|---------|----------------|
| Client email | 丁寧語 + 謙譲語 (your actions) + 尊敬語 (their actions) |
| Client phone | Same; add かしこまりました |
| Internal email to 部長 | 丁寧語; 尊敬語 on their actions optional |
| Internal Slack | 丁寧語 (ます); keigo optional |
| Incident report to client | Full formal + apology block |

## 10. Practice checklist

1. Convert five verbs from [Common words](i-common-words.md) to **いたします** form (謙譲).
2. Write one **お世話になっております** email asking for ご確認.
3. Apologize for a delay using **恐れ入りますが** + reason + **ご連絡いたします**.
4. Reply to a client with **承知いたしました** (not bare はい).
5. Never use **ご苦労様** upward — use **お疲れ様** instead.
