Getting started — Intro: Installation
What you need on your machine to work with this notes repository locally.

## 1. Git

Install **[Git](https://git-scm.com/downloads)** so you can clone, branch, commit, and push changes.

Check:

```text
git --version
```

## 2. Editor / Cursor

Use **Cursor** (or VS Code) with Markdown preview if you want to edit notes comfortably.

## 3. Clone the repo

```text
git clone https://github.com/<owner>/<repo>.git
cd <repo>
```

Replace **`<owner>`** / **`<repo>`** with your GitHub path.
